
# Cloud Act (Regulation)
## Description
The Clarifying Lawful Overseas Use of Data Act (Cloud Act) aims to clarify the legal framework for cross-border data access and sharing.


## Documentation
The Clarifying Lawful Overseas Use of Data Act (Cloud Act) is a
         regulation that aims to clarify the legal framework for cross-border
         data access and sharing. It establishes rules for the access and
         disclosure of data stored by service providers in foreign countries,
         and provides a legal basis for law enforcement agencies to request
         data from service providers. The Cloud Act applies to service
         providers that store data in the United States, as well as to foreign
         service providers that have a substantial connection to the United
         States.
